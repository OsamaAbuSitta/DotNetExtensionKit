# ExtensionKit

ExtensionKit is a lightweight .NET library that provides a curated set of extension methods to simplify everyday coding tasks. Built on .NET Standard 2.0, it works across .NET Framework, .NET Core, and .NET 5+ projects.

## Feature Highlights

### String Extensions

Manipulate and inspect strings with methods for truncation, case conversion (camelCase, snake_case, kebab-case, Title Case), masking, Base64 encoding/decoding, whitespace handling, and content extraction (digits, letters, slugs).

### DateTime Extensions

Work with dates using boundary helpers (start/end of day, week, month, year), age calculation, weekend/weekday checks, and date range validation.

### Collection Extensions

Enhance LINQ workflows with null-safe checks, batching, `ForEach` with index, `DistinctBy` key selector, safe element access with custom defaults, and dictionary helpers.

### General Extensions

Utility methods for null checking, safe casting, conditional returns, value transformation, range checks, set membership, and generic method invocation via reflection.
